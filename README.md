# BizLister (Laravel 8.4)

Migração do Flippy BizLister para Laravel 8.4.

## Setup local
- APP_URL=http://localhost/TECINFOSP/bizlister-laravel/public
- DB_DATABASE=flippy_bizlister_v2
- DB_USERNAME=root
- DB_PASSWORD=

## Passos
composer install
php artisan key:generate
npm install
npm run dev

Acesse: /public (XAMPP)
